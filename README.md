# AI-related

AI 相关项目与技能集合，通过 git submodule 管理。

## 子项目

| 子模块 | 描述 | 语言 |
|--------|------|------|
| [skills](https://github.com/github-yousen/skills) | 个人可复用 Agent 技能的公开仓库 | Python |
| [doubao-skills](https://github.com/github-yousen/doubao-skills) | 豆包 AI 平台飞书生态 Skill 集合 | Python |
| [agent-harness-comparison](https://github.com/github-yousen/agent-harness-comparison) | AI Agent Harness 技术评测 | TypeScript |

## 克隆

```bash
git clone --recurse-submodules https://github.com/github-yousen/AI-related.git
```
